# Employee Management System (EMS) - Angular + Spring Boot

## Overview
This is a CRUD-based Employee Management System built with Angular frontend and Spring Boot backend. This project serves as a learning foundation to strengthen full-stack Java development fundamentals and pave the way to becoming a proficient full-stack developer.

## What's Inside
- **Frontend**: Angular application with modern UI components
- **Backend**: Spring Boot REST API with comprehensive employee management
- **Database Operations**: Full CRUD functionality (Create, Read, Update, Delete)
- **Advanced Features**:
  - Pagination for large datasets
  - Batch processing capabilities
  - Advanced filtering options
  - Search functionality

## Tech Stack
- **Frontend**: Angular, TypeScript, HTML5, CSS3
- **Backend**: Spring Boot, Java
- **Database**: My SQL
- **Build Tools**: Maven, npm
